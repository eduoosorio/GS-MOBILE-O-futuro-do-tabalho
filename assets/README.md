# Assets

Este diretório é para os assets do aplicativo (ícones, splash screen, etc.).

## Assets Opcionais

O Expo usará assets padrão se os arquivos abaixo não existirem. O aplicativo funcionará normalmente sem eles.

Se desejar personalizar, você pode adicionar:

- `icon.png` - Ícone do aplicativo (1024x1024px)
- `splash.png` - Tela de splash (1284x2778px recomendado)
- `adaptive-icon.png` - Ícone adaptativo para Android (1024x1024px)
- `favicon.png` - Favicon para web (48x48px)

Você pode gerar esses assets usando ferramentas online ou criar manualmente.
